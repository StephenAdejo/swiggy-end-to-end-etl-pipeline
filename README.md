# Swiggy End-to-End ETL Pipeline & Analytics Project

## Overview
A full-stack data analytics project built on real Swiggy food delivery sales data.
The pipeline transforms raw CSV data into a production-grade star schema using 
layered ETL architecture, with data quality reporting, deduplication, and an 
analytics layer designed to power a Power BI dashboard.

Built from real debugging experience — including diagnosing and fixing a join 
explosion problem that led to deeper analytics engineering practices.
